# Prompt: CV and Report

Role: senior data analyst + practical ML engineer.

Goal
- Define cross-validation and reporting outputs that are verifiable.

Data
- data/raw/train.csv
- data/raw/test.csv

Constraints
- Treat conclusions as hypotheses until validated by CV.
- Keep outputs reproducible and versioned.

Tasks
1) Recommend a CV strategy (k-fold, stratified, etc.) and justify.
2) Define the report structure: metrics, feature importance, error analysis.
3) List required artifacts (configs, metrics files, plots).
4) Provide a checklist of verification steps before final submission.

Output format
- CV plan: concise steps with parameters.
- Report sections: bullet list with expected evidence.
- Verification checklist: must-pass items.
